# Visão do projeto NoteOs

A NoteOs nasce como uma distro baseada em Arch Linux com foco em um hardware específico: Samsung Galaxy Book2.

A meta não é competir com Arch, CachyOS, EndeavourOS ou Fedora. A meta é entregar uma experiência pronta, estável e otimizada para o Galaxy Book2.

## Princípios

- Upstream first
- Poucos patches próprios
- Configurações explícitas
- Reprodutibilidade
- Rollback simples
- Foco em estabilidade antes de performance extrema
- Documentação desde o início

## Fora do escopo inicial

- Linux From Scratch como base principal
- Vários desktops oficiais
- Suporte universal a qualquer notebook
- Kernel altamente modificado
- Repositório gigante próprio
